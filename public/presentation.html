Motivation & Problemstellung

Das Problem

  • Streaming-Überfluss: Netflix, Prime, Disney+, Apple TV+...
  • "Was habe ich schon gesehen?"
  • "Wo kann ich das streamen?"
  • Keine zentrale Übersicht
  • Empfehlungen nur plattformbasiert
Die Frage: Wie behalte ich den Überblick über mein Entertainment?

Feature: Hidden Gems & Discovery

Hidden Gems Algorithmus

  • Kritisch gelobt: Rating 7-9
  • Wenig bekannt: 100-1000 Votes
  • Qualität abseits des Mainstreams

Erweiterte Suche

  • Multi-Filter: Genre, Rating, Laufzeit
  • Streaming nach Region
  • Sortierung nach Beliebtheit
Beispiel: Italienische Thriller aus den 70ern auf MUBI mit Rating ≥ 7.5

Datenbank Schema

Kern-Tabellen

  • users - Benutzer + Präferenzen
  • userMovies - Film/Serien-Tracking
  • userBooks - Bücher-Tracking
  • userEpisodes - Episoden
  • reviews - Bewertungen
  • userStats - Statistiken (JSONB)

Performance

  • Indizes auf häufige Queries
  • JSONB für flexible Analytics
  • Batch-Ops gegen N+1
  • Metadaten gecached

API Integrationen

TMDB

  • Filme & Serien Daten
  • Cast, Crew, Keywords
  • Streaming-Verfügbarkeit
  • Empfehlungen

Hardcover

  • Bücher via GraphQL
  • Autoren & Cover
  • Beschreibungen
Smart Caching: Metadaten einmal abrufen → in DB speichern → API-Calls reduziert
